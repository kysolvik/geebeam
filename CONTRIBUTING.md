# Contributing Guidelines

Hi! Thanks for being interested in contribution to `geebeam`. We welcome all kinds of contributions, and there's lots of ways for people to help out. Some things you can help with:

- Add new features: Check out [issues](https://github.com/kysolvik/geebeam/issues) for ideas.
- Submit bug report or feature request: Create an issue if you encounter a bug or have a suggestion for a new feature.
- Write new examples: We'd love to have more examples of how `geebeam` can be used.

The most important rule is to please be considerate and respectful of others. For more info, see our [code of conduct](https://github.com/kysolvik/geebeam/blob/main/CODE_OF_CONDUCT.md).
